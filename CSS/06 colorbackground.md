## color

1. 키워드 사용 (red, blue, black, fransparent 등...)

2. RGB 3차원 좌표계 사용

   - 16진수 표기법 HEX(#000000)

     RGB 웹 (0, 0, 0) grba(0, 0, 0, 0.4) -> a는 투명도 소수점으로 조정

   - CMYK 출력, 인쇄물 (0%, 0%, 0%, 0%)

3. opacity 불투명도 설정

   기본값을 1으로 한다. 0.0이상 1.0이하로 조정

   0으로 지정하면 요소가 완전히 투명해져 보이지 않음.
